## Device Support

Have updated iPhone, but don't want to update Xcode? Forget DeviceSupport path every time?


## Installation

### Automatic

Run the command in your terminal:
```bash
curl -fsSL https://raw.githubusercontent.com/artemnovichkov/device-support/master/install.sh | sh
```

### Manual

1. Clone the repo
2. Unzip `11.4 (15F5061c).zip` 
3. Copy to `/Applications/Xcode.app/Contents/Developer/Platforms/iPhoneOS.platform/DeviceSupport`.
